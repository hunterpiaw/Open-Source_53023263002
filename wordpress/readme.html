รุ่น 3.5

ระบบที่ใช้เผยแพร่เรื่องราวส่วนตัวของคุณ

อย่างแรกก่อนสิ่งอื่น

ขอกล่าวยินดีต้อนรับ เวิร์ดเพรสเป็นโปรเจคที่พิเศษมากสำหรับผม นักพัฒนาและผู้สนับสนุนทุกคนต่างได้สรรค์สร้างความเป็นเลิศเข้าด้วย และพวกเราร่วมกันสร้างบางสิ่งที่สวยงามที่ผมภูมิใจที่ได้เป็นส่วนหนึ่งในการสร้างสิ่งนี้ เวลากว่าหลายพันชั่วโมงที่ทุ่มเทไปกับเวิร์ดเพรส และเราก็อุทฺสเวลาเพื่อทำให้เวิร์ดเพรสดีขึ้นทุกวัน ขอบคุณที่เลือกเวิร์ดเพรสเป็นส่วนหนึ่งของชีวิตคุณ

— Matt Mullenweg

การติดตั้ง: การติดตั้งภายใน 5 นาทีที่มีชื่อเสียงของเรา

  1. แตกซิปไฟล์ติดตั้งลงในไดเรคทอรีที่ต้องการติดตั้ง
  2. เปิดไฟล์ /wp-admin/install.php ในเบราเซอร์ของคุณ จะปรากฎขั้นตอนการตั้งค่าที่ต้องการสำหรับไฟล์ wp-config.php ด้วยข้อมูลเชื่อมต่อฐานข้อมูลของคุณ
    1. ถ้ามีเหตุผลบางอย่างที่ทำให้มันไม่ทำงาน ไม่ต้องกังวล มันอาจจะทำงานไม่ได้บางเว็บโฮสท์ ให้เปิดไฟล์ wp-config-sample.php ด้วยตัวแก้ไขตัวอักษรอย่าง WordPad หรือโปรแกรมคล้าย ๆ กัน และใส่ข้อมูลเชื่อมต่อฐานข้อมูลของคุณ
    2. บันทึกไฟล์เป็น wp-config.php และอัปโหลดมัน
    3. เปิด wp-admin/install.php ในเบราว์เซอร์ของคุณ
  3. เมื่อไฟล์ตั้งค่าถูกตั้งค่าแล้ว ตัวติดตั้งจะสร้างตารางฐานข้อมูลที่ต้องการสำหรับบล็อกของคุณ ถ้ามีความผิดพลาด โปรดเช็คไฟล์ wp-config.php ของคุณและลองอีกครั้ง ถ้ายังไม่ได้ ให้ไปที่ เวิร์ดไทยฟอรั่ม พร้อมด้วยข้อมูลเกี่ยวกับความผิดพลาดมากที่สุดที่คุณสามารถรวบรวมได้
  4. ถ้าคุณไม่ใส่รหัสผ่าน ให้จดรหัสผ่านที่ระบบสร้างให้กับคุณ ถ้าคุณไม่ได้ใส่ชื่อผู้ใช้ ชื่อผู้ใช้จะเป็น admin
  5. ระบบควรจะนำคุณมายังหน้า เข้าสู่ระบบ login page เข้าสู่ระบบด้วย username admin และรหัสผ่านที่ได้ระหว่างการติดตั้ง ซึ่งคุณมาสามารถเปลี่ยนมันภายหลังได้ที่หน้า Profile

การอัปเดต

โดยใช้ตัวอัปเดตอัตโนมัติ

ถ้าคุณกำลังอัปเดตจากรุ่น 2.7 หรือสูงกว่า คุณสามารถใช้ตัวอัปเดตัตโนมัติได้

  1. เปิดหน้า wp-admin/update-core.php ในเบราเซอร์ของคุณและทำตามคำแนะนำ
  2. คุณคิดว่ามีขั้นตอนอีกหรือ? ไม่ล่ะ มันมีแค่นี้เอง

การอัปเกรดด้วยตัวเอง

  1. ก่อนที่คุณจะทำการอัปเกรดสิ่งใด พึงระวังเสมอว่าคุณได้ทำการแบคอัปไฟล์ทุกอย่างไว้ และสิ่งที่คุณได้ทำการแก้ไขอย่างเช่นไฟล์ index.php
  2. ลบไฟล์ WP เก่าของคุณและบันทึกไฟล์บางส่วนที่คุณได้ทำการแก้ไขไป
  3. อัปโหลดไฟล์ใหม่ทั้งหมด
  4. ใช้เบราเซอร์ของคุณเข้าไปที่ /wp-admin/upgrade.php

การเปลี่ยน Template

ถ้าคุณได้ทำการดัดแปลง templates ของคุณ คุณอาจจะต้องทำการเปลี่ยนบางอย่างก่อน ถ้าคุณต้องการแปลง template ของคุณที่เป็นรุ่น 1.2 หรือก่อนหน้าเราได้ทำคำแนะนำพิเศษสำหรับคุณ

การย้ายมาจากระบบอื่น

เวิร์ดเพรสสามารถ นำเข้าจากหลาย ๆ ระบบ อย่างแรกคุณต้องติดตั้งเวิร์ดเพรสให้สามารถทำงานได้อย่างที่ระบุไว้ด้านบน ก่อนที่จะใช้เครื่องมือนำเข้าของเรา

ความต้องการของระบบ

  • PHP รุ่น 5.2.4 หรือสูงกว่า
  • MySQL รุ่น 5.0หรือสูงกว่า

ระบบที่แนะนำ

  • mod_rewrite Apache module
  • ลิงก์ไปที่ http://th.wordpress.orgจากเว็บไซต์ของคุณ

แหล่งความรู้ออนไลน์

ถ้าคุณมีปัญหาที่ไม่ได้ถูกระบุอยู่ในเอกสารนี้ มีแหล่งความรู้ออนไลน์อื่น ๆ ที่คุณสามารถหาความรู้จากที่นั่นได้:

เอกสารประกอบเวิร์ดเพรส
หน้ารวมเอกสารประกอบเป็นดั่งสารานุกรมของทุกอย่างที่เกี่ยวข้องกับเวิร์ดเพรส ที่นั่นมีแหล่งข้อมูลจำนวนมากเกี่ยวกับเวิร์ดเพรส
บล็อคการพัฒนาเวิร์ดเพรส
ที่นี่จะรวมไฟล์และข้อมูลอัพเดตล่าสุดเกี่ยวกับเวิร์ดเพรส แนะนำให้คุณ Bookmark หน้านี้และมาเยี่ยมบ่อย ๆ
โลกของเวิร์ดเพรส
โลกของเวิร์ดเพรสนั้นจะรวบรวมเรื่องต่าง ๆ จากบล็อคเวิร์ดเพรสจากที่ต่าง ๆ
ชุมชนช่วยเหลือของเวิร์ดเพรส
ถ้าคุณได้ค้นหาจากทุกที่และไม่สามารถหาคำตอบที่ตรงใจคุณได้ ชุมชนช่วยเหลือของเวิร์ดเพรสนั้น มีคนจำนวนมากที่พร้อมจะช่วยเหลือและตอบคำถามให้คุณ เพื่อให้พวกเขาสามารถช่วยแก้ปัญหาคุณได้อย่างถูกต้อง คุณควรจะรวบรวมรายละเอียดเกี่ยวกับปัญหาของคุณให้มากที่สุด
IRC ของเวิร์ดเพรส
ในที่สุด เราก็มีช่องทางสำหรับการคุยออนไล์ ที่จะใช้เป็นที่สนทนาสำหรับผู้ใช้เวิร์ดเพรส และช่วยเหลือสำหรับการตอบคำถามในบางหัวข้อ คุณสามารถเข้า IRC ของเวิร์ดเพรสได้ที่ (irc.freenode.net #wordpress)
ชุมชนเวิร์ดเพรสไทย
แหล่งข้อมูลเกี่ยวกับ การติดตั้ง การใช้งาน ปลั๊คอิน การตั้งค่าต่าง ๆ และไฟล์ภาษาไทยของเวิร์ดเพรส Wordpress Thai

XML-RPC และ Atom Interface

คุณสามารถส่งเรื่องมายังบล็อคเวิร์ดเพรสของคุณด้วยเครื่องมือเช่น Windows Live Writer, Ecto, Bloggar, Radio Userland (ซึ่งหมายถึงคุณสามารถใช้งาน Radio's email-to-blog feature), NewzCrawler และเครื่องมืออื่น ๆ ที่สนับสนุน Blogging APIs! :) คุณสามารถอ่านรายละเอียดเพิ่มเติมได้ที่ เอกสารประกอบการใช้งาน XML-RPC

ส่งเรื่องผ่านอีเมล์

คุณสามารถส่งเรื่องผ่านทางอีเมล์ การตั้งค่านั้นให้ไปที่หน้าจอตัวเลือก "การเขียน (Writing)" และใส่ข้อมูลการเชื่อมต่อสำหรับบัญชี POP3 ลับของคุณ แล้วคุณต้องทำการตั้งค่าไฟล์ wp-mail.php เพื่อรันคำสั่งเป็นเวลาให้เช็คในเมล์บ็อกซ์ว่ามีเรื่องใหม่หรือไม่ คุณสามารถทำโดยใช้ Cron-jobs, หรือถ้าโฮสท์ของคุณไม่สนับสนุน คุณสามารถดูบริการของ website-monitoring หลาย ๆ ที่ และตั้งค่าให้บริการเหล่านั้นเช็ค URL ของ wp-mail.php ของคุณ

การเขียนเรื่องนั้นง่ายมาก: อีเมล์ใด ๆ ที่ส่งเข้ามาในเมล์บ็อกซ์ที่คุณตั้งค่าไว้จะถูกเขียนเป็นเรื่อง พร้อมด้วย subject เป็นหัวข้อ ดังนั้นคุณควรจะเก็บอีเมล์นี้เป็นความลับ และ script จะทำการ ลบ อีเมล์หลังจากที่ส่งเรื่องเสร็จแล้ว

บทบาทผู้ใช้

เราได้จำกัดสิทธิ์ผู้ใช้ไว้เป็นระดับขั้นเพื่อ ให้บทบาทของผู้ใช้แต่ละคนนั้นยืดหยุ่นเหมาะสมกับระบบเวบแบบ 2.0 คุณสามารถ อ่านเอกสารเพิ่มเติมเกี่ยวกับบทบาทและความสามารถได้ที่นี่

หมายเหตุสุดท้าย

  • ถ้าคุณมีข้อเสนอแนะ ความเห็นต่าง ๆ หรือคุณพบว่าเวิร์ดเพรสมีความผิดพลาด เรายินดีรับฟังคุณที่ ชุมชนช่วยเหลือ.
  • ในตอนนี้ เวิร์ดเพรสมีระบบ API สำหรับปลั๊คอินที่มีความสามารถมาก ทำให้การเขียน code เพิ่มเติมทำให้ง่าย ถ้าคุณเป็นผู้พัฒนาและสนใจในเวิร์ดเพรส กรุณาดูที่ เอกสารประกอบเรื่องปลั๊กอิน ส่วนมากในการพัฒนาคุณไม่ควรจะแตะ code ส่วนที่เป็นระบบหลัก

ส่งต่อความรักของคุณที่มีต่อเวิร์ดเพรส

เวิร์ดเพรสนั้นไม่มีเงินทุนมากมายที่จะทำการตลาดหลักหลายล้าน US$ หรือมีผู้สนับสนุนที่มีชื่อเสียง แต่เรามีบางอย่างที่มีค่ามากกว่า นั่นคือคุณ ถ้าคุณชื่นชอบในเวิร์ดเพรส กรุณาบอกต่อกับเพื่อนของคุณ ช่วยตั้งค่าเริ่มต้นให้กับคนที่มีความรู้น้อยกว่าคุณ หรือเขียนหาผู้สื่อข่าวต่าง ๆ ที่ไม่สนใจเรา

เวิร์ดเพรสนั้นพัฒนามาจาก b2/cafélog ซึ่งผู้พัฒนาเดิมคือ Michel V สิ่งนี้ได้ถูกสืบทอดต่อโดย ผู้พัฒนาเวิร์ดเพรส ถ้าคุณต้องการสนับสนุนเวิร์ดเพรส กรุณาคิดถึง การบริจาคให้แก่เรา
